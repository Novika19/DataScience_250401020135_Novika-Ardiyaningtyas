## Cara Menjalankan Notebook

### Menggunakan Google Colab

1. Buka repository GitHub.
2. Pilih file notebook (.ipynb) yang ingin dijalankan.
3. Unduh notebook atau buka menggunakan Google Colab.
4. Pilih **File → Upload Notebook** untuk mengunggah file notebook ke Google Colab.
5. Jalankan seluruh sel secara berurutan menggunakan **Runtime → Run All**.
6. Pastikan seluruh sel berjalan tanpa error dan menghasilkan output yang sesuai.
